# Resume-Analyzer
A simple AI based project that analyzes resume text, identifies skills, and suggests improvements.

# Problem
Students often do not know which skills are missing in their resumes.

# Solution
This project analyzes resume text and identifies skills present and missing.

#Features
1. Detects skills from resume
2. Shows missing skills
3. Calculates resume score
4. Gives suggestions

# Technologies Used
 Python

# How to Run
1. Run: python main.py
2. Paste your resume text

# Output
Displays skills found, missing skills, score, and suggestions.
